# ModeraBackendToolsBundle

## Installation

Add this dependency to your composer.json:

    "modera/backend-tools-bundle": "dev-master"

Update your AppKernel class and add this:

    new Modera\BackendToolsBundle\ModeraBackendToolsBundle(),
